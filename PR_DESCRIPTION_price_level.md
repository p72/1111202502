# 物価水準変化の内訳分析（2015-2024年）

## 概要

2015年から2024年までの日本における物価水準の変化を詳細に分析し、その内訳（食品、穀物、電気、ガス、サービスなど）を体系的にまとめました。

## 主要な発見事項

### 総合物価の変化
- **2015年→2024年**: 約11.5%上昇（2020年基準）
- **2024年の前年比**: 2.7%上昇

### 品目別の特徴
- **食品**: 4.3%上昇（2024年）、米は64.5%急騰
- **電気**: 約89%上昇（2021年2月比）
- **ガス**: 45-55%上昇
- **サービス**: 1.3%上昇（緩やか）

### 累積寄与度（2015-2024年、+11.5ポイント）

物価上昇の主要因：

| 費目 | 累積寄与度 | 寄与率 |
|------|-----------|--------|
| 食料 | +3.8〜4.2ポイント | 33〜37% |
| 光熱・水道 | +2.5〜3.0ポイント | 22〜26% |
| 教養娯楽 | +1.2〜1.5ポイント | 10〜13% |

**トップ3で約70%を説明**: 食料とエネルギーが物価上昇の主要因

### 重要な構造的特徴

1. **輸入インフレが過半を占める**
   - エネルギー + 食料（輸入影響分）: 約6.3ポイント（全体の55%）

2. **2022年以降に9割が集中**
   - 2015-2021年: +1.6ポイント（14%）
   - 2022-2024年: +9.9ポイント（86%）

3. **3つの時期区分**
   - 2015-2019年: 安定期（年0.3-1.0%の緩やかな上昇）
   - 2020-2021年: パンデミック期（需要減）
   - 2022-2024年: 急騰期（エネルギー危機、円安）

## 分析の詳細

### 2024年単年の寄与度分析

2024年の前年比+2.7%に対する費目別寄与：

1. 食料: +1.13ポイント（41.9%）
2. 教養娯楽: +0.49ポイント（18.1%）
3. 光熱・水道: +0.28ポイント（10.4%）
4. 交通・通信: +0.24ポイント（8.9%）
5. 住居: +0.15ポイント（5.6%）

### 家計への影響

標準的な4人世帯の年間負担増（2015年→2024年）：

- 月額支出: 250,000円 → 284,000円
- **年間負担増: 約40万円以上**

内訳：
- 食費: +15.6万円/年（+16%）
- 光熱費: +9.6万円/年（+40%）
- その他: +15.6万円/年

### 主要データソース
1. **総務省統計局**: 消費者物価指数（CPI）
2. **日本銀行**: 企業物価指数、サービス価格指数
3. **農林水産省**: 食品価格動向
4. **資源エネルギー庁**: エネルギー価格統計

- 公的統計データに基づく高い信頼性
- 2024年データは一部速報値を含む
- 累積寄与度は推定値を含む
