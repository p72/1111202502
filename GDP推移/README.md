# GDP推移 - 日本の名目GDPと長期金利の推移分析

## 📁 フォルダ概要

このフォルダには、日本の名目GDPと長期金利（10年国債利回り）の1980年から2024年までの45年間の推移データと分析結果が含まれています。

## 📊 生成ファイル一覧

### データファイル

1. **japan_gdp_interest_data.csv**
   - 全データをCSV形式で保存
   - 列: 年度、名目GDP（兆円）、長期金利（%）
   - 用途: Excelやスプレッドシートでの分析に最適

2. **japan_gdp_interest_data.json**
   - 全データとエ計サマリーをJSON形式で保存
   - データと統計情報を含む構造化フォーマット
   - 用途: プログラムでの読み込み、APIとの連携に最適

### レポート・分析文書

3. **analysis_report.md**
   - 📄 **メインレポート** - 詳細な分析レポート
   - 内容:
     - エグゼクティブサマリー
     - 年次データ一覧（全45年分）
     - 時期別分析（バブル期、失われた10年、アベノミクス期など）
     - 統計分析
     - テキストチャート
     - 主要な洞察と結論
   - 推奨: まずこのファイルを読むことをお勧めします

4. **statistical_summary.txt**
   - 統計サマリーのテキスト版
   - GDP・金利の基本統計量
   - 相関分析結果
   - 主要経済イベント

### 可視化

5. **visualization.html**
   - 📈 **インタラクティブグラフ** - ブラウザで開いて表示
   - 含まれるグラフ:
     - 名目GDPの推移（折れ線グラフ）
     - 長期金利の推移（折れ線グラフ）
     - GDP・金利の2軸グラフ
     - GDP成長率（棒グラフ）
     - GDP・金利の散布図（時期別）
   - 推奨: ブラウザで開いてインタラクティブに探索

### スクリプト

6. **create_data.py**
   - データ生成スクリプト（標準ライブラリのみ使用）
   - CSV、JSON、統計レポートを生成

7. **japan_gdp_interest_analysis.py**
   - 高度な分析・可視化スクリプト（pandas、matplotlib使用）
   - グラフ画像ファイルを生成（環境に依存）

## 📈 主要な発見

### 🔑 キーファインディング

1. **名目GDPの長期成長**
   - 45年間で約2.7倍に増加（249.4兆円 → 680.5兆円）
   - 増加額: 431.1兆円（+172.9%）

2. **長期金利の構造的低下**
   - 9.22% → 1.15%（-8.07ポイント）
   - 2016年にはマイナス金利（-0.02%）を経験

3. **強い負の相関**
   - 相関係数: **-0.849**
   - GDPが増加する中で金利は持続的に低下

4. **政策転換点（2024年）**
   - 30年ぶりの本格的な金利上昇トレンド
   - 2022-2024年で金利が約1ポイント上昇

## 🗓️ 重要な経済イベント

| 時期 | イベント | GDP変化 | 金利変化 |
|------|---------|---------|---------|
| 1980年代後半 | バブル経済 | 急成長 | 高金利維持 |
| 1991-2002 | 失われた10年 | 停滞 | 大幅低下 |
| 2008-2009 | リーマンショック | -7.2% | 低位安定 |
| 2013 | アベノミクス開始 | 緩やか回復 | 超低金利継続 |
| 2016 | マイナス金利導入 | 継続成長 | マイナス転換 |
| 2020 | COVID-19 | -2.4% | 超低金利維持 |
| 2022-2024 | 金利正常化 | 継続成長 | 上昇転換 |

## 📊 統計サマリー

### 名目GDP
- **平均値**: 552.9兆円
- **最大値**: 680.5兆円（2024年）
- **最小値**: 249.4兆円（1980年）
- **年平均成長率**: 約2.3%

### 長期金利
- **平均値**: 2.26%
- **最高値**: 9.22%（1980年）
- **最低値**: -0.02%（2016年）
- **マイナス金利期間**: 2016年、2019年

## 🔍 時期別特徴

### 第1期: バブル期 (1980-1990)
- GDP年平均成長率: +7.5%
- 金利: 高水準（6-9%台）

### 第2期: 失われた10年 (1991-2000)
- GDP年平均成長率: +1.5%
- 金利: 急速に低下（6% → 1%台）

### 第3期: デフレと低成長 (2001-2010)
- GDP年平均成長率: +0.3%（ほぼ横ばい）
- 金利: 1%前後で推移

### 第4期: アベノミクス (2011-2020)
- GDP年平均成長率: +1.2%
- 金利: ゼロ近傍、マイナス金利導入

### 第5期: コロナ後と金利正常化 (2021-2024)
- GDP年平均成長率: +2.7%
- 金利: 上昇トレンドへ転換（0.1% → 1.15%）

## 📚 データソース

- **名目GDP**: 内閣府 国民経済計算
- **長期金利**: 財務省・日本銀行（10年国債利回り）
