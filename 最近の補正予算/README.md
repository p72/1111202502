# 最近の補正予算 - 詳細分析プロジェクト

本フォルダには、日本の最近の補正予算（令和4年度〜令和6年度）に関する詳細な調査・分析資料が含まれています。

## 📁 フォルダ構成

### 📄 主要ドキュメント

- **分析レポート.md** - 補正予算に関する包括的な分析レポート
  - 総額推移の分析
  - 支出項目の詳細内訳
  - 基金の設置と運用状況
  - 執行状況と繰越の分析
  - 未執行残高の推移
  - 課題と政策提言

### 📊 データファイル

- **補正予算データ.csv** - 年度別の補正予算額、執行率、繰越額などの数値データ
- **基金一覧.csv** - 補正予算で設置された主要基金のリスト

### 📈 可視化グラフ

1. **01_総額推移.png** - 補正予算の総額推移（2016-2024年度）
   - 年度別の補正予算規模
   - COVID-19期の影響の可視化
   - 最近の平均値との比較

2. **02_執行状況分析.png** - 執行状況の分析
   - 令和4年度の執行・繰越・未執行の内訳（円グラフ）
   - 年度別の執行率と繰越率の推移（棒グラフ）

3. **03_支出項目内訳.png** - 支出項目の詳細内訳
   - 令和6年度補正予算の項目別内訳（横棒グラフ）
   - 過去3年間の支出構成の推移（積み上げ棒グラフ）

4. **04_基金分析.png** - 主要基金の分析
   - 基金の種類別の規模（円グラフ）

5. **04_基金分析_type.png** - 基金の特性分析
   - 執行期間別の基金数と平均規模

6. **05_残高推移.png** - 未執行残高の推移
   - 累積補正予算額と累積執行額の推移
   - 未執行残高の年次推移

## 🔍 主要な発見事項

### 1. 補正予算の規模
- 令和4年度第2次: **28.9兆円**（物価高対策・経済安保）
- 令和5年度: **13.2兆円**（物価高対策・投資促進）
- 令和6年度: **13.9兆円**（物価高対策・投資促進）

### 2. 執行状況の課題
- 令和4年度の経済対策事業18.86兆円のうち、**46%が翌年度に繰越**
- 執行率は近年改善傾向にあるものの、依然として課題が残る

### 3. 基金の活用拡大
- 半導体支援（2.0兆円）、GX（1.5兆円）など大規模基金を設置
- 複数年度にわたる柔軟な支出を可能にする一方、透明性の課題も

### 4. 未執行残高の累積
- 2024年時点で累積未執行残高は**約23兆円**
- 財政規律の観点から課題

## 📊 グラフの見方

各グラフには英語と日本語の両方でタイトルが付けられています。日本語フォントの制約により、一部の文字が正しく表示されない場合がありますが、データと分析内容は正確です。

## 🛠️ グラフの再生成方法

グラフを再生成する場合は、以下のコマンドを実行してください：

```bash
cd "最近の補正予算"
python 01_総額推移.py
python 02_執行状況分析.py
python 03_支出項目内訳.py
python 04_基金分析.py
python 05_残高推移.py
```

**必要な環境**:
- Python 3.7以上
- matplotlib
- numpy

## 📚 データソース

1. 財務省の補正予算関連資料
2. 会計検査院の決算検査報告
3. 各府省の予算資料
4. 国会審議資料
5. 経済分析レポート
