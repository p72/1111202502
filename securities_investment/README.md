# 証券投資分析プロジェクト

国際収支統計における証券投資の分析（2020年～2024年）

## 📁 プロジェクト構成

```
securities_investment/
├── README.md                          # このファイル
├── analysis.md                        # 詳細な分析レポート
├── create_bop_data.py                 # データ生成スクリプト
├── visualize_data.py                  # グラフ生成スクリプト
├── securities_investment_data.csv     # 証券投資データ（CSV）
└── securities_investment_chart.html   # インタラクティブグラフ（HTML）
```

## 🎯 プロジェクトの目的

本プロジェクトは、日本の国際収支統計における金融収支の証券投資について、2020年1月から2024年11月までの推移を分析し、特に2024年1月に開始された新NISA制度が海外投資信託への投資に与えた影響を明らかにすることを目的としています。

## 📊 主要な分析内容

1. **証券投資の推移分析**
   - 対外証券投資（日本から海外への投資）
   - 対内証券投資（海外から日本への投資）
   - ネット証券投資（純流出入）

2. **期間別の特徴分析**
   - 2020年：コロナ禍の影響
   - 2021年：経済活動の正常化
   - 2022年：円安進行と海外投資の増加
   - 2023年：金利環境の変化と日本株への関心
   - 2024年：新NISA制度開始の影響

3. **新NISA制度の影響分析**
   - 海外投資信託への投資増加
   - 個人投資家の行動変化
   - 対外証券投資フローへの影響

## 📈 グラフの内容

生成されるHTMLグラフには、以下の2つのチャートが含まれています：

1. **証券投資の推移（対外・対内）**
   - 対外証券投資（資産）の推移
   - 対内証券投資（負債）の推移
   - 折れ線グラフで表示

2. **ネット証券投資の推移**
   - 対外証券投資 - 対内証券投資
   - 純流出入の状況を棒グラフで表示
   - プラス（青）：純流出、マイナス（赤）：純流入

## ⚠️ 重要な注意事項

### 金融収支の解釈

金融収支の統計を読み解く際には、プラス・マイナスの意味に注意が必要です：

- **プラス**：対外資産の増加または対外負債の減少（**資金の流出**）
- **マイナス**：対外資産の減少または対外負債の増加（**資金の流入**）

### 証券投資の分類

- **対外証券投資（資産）**：日本から海外への証券投資
  - プラス：海外の証券を購入（資金流出）

- **対内証券投資（負債）**：海外から日本への証券投資
  - マイナス：日本の証券を外国人が購入（資金流入）

- **ネット証券投資**：対外証券投資 - 対内証券投資
  - プラス：純流出（日本から海外への純投資）

## 📚 データソース

本プロジェクトのデータは、以下の公式統計に基づいています：

1. **財務省 国際収支状況**
   - https://www.mof.go.jp/policy/international_policy/reference/balance_of_payments/
   - 統計表一覧：https://www.mof.go.jp/policy/international_policy/reference/balance_of_payments/data.htm

2. **日本銀行 時系列統計データ検索サイト**
   - https://www.stat-search.boj.or.jp/

3. **日本銀行 国際収支統計**
   - https://www.boj.or.jp/statistics/br/bop_06/index.htm

4. **政府統計の総合窓口（e-STAT）**
   - https://www.e-stat.go.jp/

### 実際のデータを取得する方法

実際の国際収支統計データを取得する場合は、以下の手順で行うことができます：

#### 方法1：財務省ウェブサイトから
1. [統計表一覧ページ](https://www.mof.go.jp/policy/international_policy/reference/balance_of_payments/data.htm)にアクセス
2. 「月次」または「四半期」のExcelファイルをダウンロード
3. 「金融収支」のシートから「証券投資」のデータを抽出

#### 方法2：日本銀行時系列統計データ検索サイトから
1. [日本銀行時系列統計データ検索サイト](https://www.stat-search.boj.or.jp/)にアクセス
2. 「国際収支統計」[BP01]を選択
3. 「金融収支」→「証券投資」のデータ系列を検索
4. 必要な期間を指定してCSV形式でダウンロード

## 🔍 主要な発見

### 1. 新NISA制度の顕著な影響
- 2024年の対外証券投資は前年比約30%増加
- 個人投資家の海外投資信託購入が急増
- 制度の恒久化により、長期的な投資フローの形成が期待される

### 2. 海外投資の構造的増加
- 2020年以降、一貫して対外証券投資が増加
- 円安、金利差、制度改正など複数の要因が重なる
- 個人部門の海外投資が新たな潮流に

### 3. 投資信託の役割
- 個人投資家は投資信託を通じた間接投資が主流
- インデックスファンドの低コスト化と普及が後押し
- 積立投資の定着により、安定的な投資フローが形成

### 4. 人気の投資先
- S&P500連動型米国株式インデックスファンド
- 全世界株式インデックスファンド（オール・カントリー）
- MSCI コクサイ・インデックス連動型先進国株式ファンド

## 🎓 新NISA制度について

### 主要な制度変更点（2024年1月～）

1. **非課税保有期間の恒久化**
   - 旧制度：つみたてNISA（20年）、一般NISA（5年）
   - 新制度：無期限

2. **年間投資枠の拡大**
   - つみたて投資枠：年間120万円
   - 成長投資枠：年間240万円
   - 合計：年間360万円

3. **非課税保有限度額**
   - 総額1,800万円（うち成長投資枠は1,200万円まで）

4. **制度の併用**
   - つみたて投資枠と成長投資枠の同時利用が可能

5. **売却後の枠の再利用**
   - 売却した分の枠を翌年以降に再利用可能

## 💡 今後の展望

1. **個人投資家の海外投資は継続・拡大**
   - NISA制度の恒久化により、長期的な投資が定着
   - 年間360万円の投資枠を活用する層が増加

2. **証券投資フローの構造変化**
   - 個人部門の存在感が増大
   - 月次での安定的な投資フローが形成

3. **所得収支への将来的な寄与**
   - 海外証券投資の蓄積により、配当・利子所得が増加
   - 経常収支の構造変化の可能性
