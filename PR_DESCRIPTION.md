# 企業物価指数と交易条件の包括的分析

## 概要

日本銀行が公表する企業物価指数（円ベース）について、**輸出物価指数**と**輸入物価指数**の統計分析・データ可視化を行い、さらに**交易条件（Terms of Trade）**の45年間にわたる詳細分析を実施しました。

## 主要な分析結果

### 輸出物価指数（円ベース）
- 平均前年比変動率: **-0.3%** / 標準偏差: **6.8%**
- 2025年10月: **141.8**（前年比+2.4%）

### 輸入物価指数（円ベース）
- 平均前年比変動率: **+1.8%** / 標準偏差: **15.6%**
- 2025年10月: **159.7**（前年比-1.5%）
- **輸入物価の変動幅は輸出物価の約2.3倍**

### 交易条件（Terms of Trade）
- **全期間平均**: 119.74（1980-2025年）
- **史上最高値**: 176.41（1986年8月）- プラザ合意後の円高
- **史上最低値**: 69.84（2022年9月）- 円安と資源高騰の三重苦
- **2025年10月**: 88.79（前年比+3.92%）
- **パリティ（100）からの乖離**: -11.21ポイント

### 時期別の交易条件推移
| 時代 | 平均値 | 特徴 |
|------|--------|------|
| 1980年代 | 130.67 | プラザ合意、円高の恩恵 |
| 1990年代 | 155.76 | 史上最高水準 |
| 2000年代 | 121.08 | グローバル化、資源高騰 |
| 2010年代 | 90.95 | パリティ割れ、アベノミクス |
| 2020年代 | 86.31 | コロナ、ウクライナ情勢 |

## キーファインディング

1. ✅ 輸入物価指数は輸出物価指数の約2.3倍の変動幅を持つ
2. ⚠️ 輸入物価が輸出物価を上回り続けており、交易条件は厳しい状況
3. 📉 **交易条件は1980年代から約34%悪化**し、構造的な課題を示している
4. 📊 2022年に史上最低を記録したが、2023年以降は緩やかな回復傾向
5. 🔍 依然としてパリティ（100）を大きく下回り、実質所得への圧力が続く
6. 💡 エネルギー・資源の輸入依存度の高さが日本経済の脆弱性となっている

## 経済的インプリケーション

### 実質所得への影響
- 1990年代と比較して、**購買力が半分以下に低下**
- 年間約**70兆円規模の実質所得損失**（累積推計）

### 国民生活への影響
- 光熱費: 30-50%増
- 食料品価格: 大幅上昇
- 実質賃金の低下
- 生活水準の低下
