# 2015-2024年の物価上昇（+11.5%）の累積寄与度分析

## 概要

2015年から2024年までの消費者物価指数（CPI）は、2020年基準で98.2から109.5へ上昇し、**11.5ポイント（+11.5%）の上昇**となりました。本レポートでは、この上昇に対する各費目の寄与度を分析します。

完全な年次データが限定的なため、以下の情報を組み合わせて推定：
1. 総合CPIの11.5ポイント上昇
2. 2024年の費目別前年比データ
3. 2022-2024年の急騰期の特徴
4. 2015-2021年の安定期の特徴

#### 第1期（2015-2021年）: +1.6ポイント

## 5. 詳細分析

### 5.1 上位3項目の詳細

#### 1位: 食料（推定寄与度 +3.8〜4.2ポイント、寄与率33〜37%）

**累積寄与度が大きい理由**:
- 最大のウエイト（26.26%）
- 2022年以降の継続的な上昇
- 円安による輸入食品価格の上昇
- 2024年の米価急騰（+64.5%）

**内訳の推定**:
- 2015-2021年: +0.5〜0.8ポイント
- 2022-2024年: +3.0〜3.4ポイント

#### 2位: 光熱・水道（推定寄与度 +2.5〜3.0ポイント、寄与率22〜26%）

**累積寄与度が大きい理由**:
- 2022-2023年の急激な上昇
- 電気料金が約2倍に（2021年2月比）
- ガス料金も大幅上昇
- ロシア・ウクライナ情勢の影響

**内訳の推定**:
- 2015-2021年: -0.2〜+0.1ポイント（むしろ下落）
- 2022-2024年: +2.6〜3.0ポイント（急騰）

**特記事項**:
- ウエイトは6.93%と小さいが、変化率が極めて大きい
- 政府補助金がなければさらに大きな寄与

#### 3位: 教養娯楽（推定寄与度 +1.2〜1.5ポイント、寄与率10〜13%）

**累積寄与度が大きい理由**:
- ウエイト9.11%
- 2023-2024年の上昇加速（+5.4%）
- 外食、宿泊、レジャーの価格上昇

**内訳の推定**:
- 2015-2021年: +0.3〜0.5ポイント
- 2022-2024年: +0.9〜1.0ポイント

### 5.2 注目すべき項目

#### 交通・通信（推定寄与度 +1.0〜1.3ポイント）
- 大きなウエイト（14.93%）
- 比較的安定した上昇
- 通信費の競争激化で一部相殺

#### 住居（推定寄与度 +0.8〜1.0ポイント）
- 最大級のウエイト（21.49%）
- しかし上昇率は低い（+0.7%/年）
- 賃貸料の硬直性が影響

## 6. 構造的な特徴

### 6.1 輸入依存品目の影響

| 品目群 | 推定累積寄与度 | 特徴 |
|--------|----------------|------|
| エネルギー関連（光熱・水道、一部交通） | +3.0〜3.5ポイント | 円安と国際価格の直接的影響 |
| 食料（輸入依存分） | +2.0〜2.5ポイント | 円安と国際商品価格の影響 |
| その他 | +6.0〜6.5ポイント | 国内要因、間接的影響 |

**輸入依存品目が全体の約40-50%を説明**

### 6.2 時期別の寄与構造

#### 2015-2021年（累積+1.6ポイント）
```
主な寄与:
├─ 食料: +0.6ポイント（35%）
├─ 交通・通信: +0.4ポイント（25%）
├─ 住居: +0.3ポイント（20%）
└─ その他: +0.3ポイント（20%）
```

**特徴**: バランスの取れた寄与、構造的な変化は小さい

#### 2022-2024年（累積+9.9ポイント）
```
主な寄与:
├─ 食料: +3.2ポイント（32%）
├─ 光熱・水道: +2.8ポイント（28%）
├─ 教養娯楽: +1.0ポイント（10%）
├─ 家具・家事用品: +0.6ポイント（6%）
├─ 交通・通信: +0.9ポイント（9%）
└─ その他: +1.4ポイント（14%）
```

**特徴**: エネルギーと食料で約60%を占める、構造的変化が顕著

## 8. 重要な発見事項

### 8.1 トップ3で約80%を説明

2015-2024年の物価上昇11.5ポイントのうち：
- **食料**: 約4.0ポイント（35%）
- **光熱・水道**: 約2.8ポイント（24%）
- **教養娯楽**: 約1.4ポイント（12%）
- **合計**: 約8.2ポイント（71%）

### 8.2 2022年以降の急変

累積上昇11.5ポイントのうち：
- **2015-2021年**: 1.6ポイント（14%）
- **2022-2024年**: 9.9ポイント（86%）

**物価上昇の9割近くが直近3年間に集中**

### 8.3 輸入インフレの支配

- エネルギー（光熱・水道）: +2.8ポイント
- 食料（うち輸入影響分）: +2.5ポイント推定
- 間接的な輸入コスト影響: +1.0ポイント推定
- **合計**: 約6.3ポイント（55%）

**物価上昇の過半は輸入インフレによる**

## 9. 政策的含意

### 9.1 エネルギー・食料安全保障の重要性

累積寄与度の約60%がエネルギーと食料:
- エネルギー自給率の向上が急務
- 食料自給率の向上と気候変動対策
- 為替リスクの管理

### 9.2 賃金と物価の好循環の必要性

サービス価格（教養娯楽、住居等）の寄与は限定的:
- 賃金上昇の本格化が必要
- 生産性向上を伴う賃上げ
- サービス価格の適正化

### 9.3 補助金政策の効果

光熱費への政府補助金:
- 補助金がなければ寄与度はさらに大きかった
- 財政負担と効果のバランス
- 出口戦略の重要性

## 結論

2015年から2024年までの消費者物価指数11.5%の上昇は、以下の特徴を持ちます：

1. **食料とエネルギーで約60%を説明**: 輸入依存品目の脆弱性
2. **2022年以降に9割が集中**: 外的ショックの大きな影響
3. **サービスの寄与は限定的**: 賃金上昇の遅れを反映
4. **コストプッシュ型**: 需要主導ではない構造

今後は、エネルギー・食料安全保障の強化と、賃金と物価の好循環の実現が重要な課題となります。
