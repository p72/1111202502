# 物価水準変化の内訳分析（2015-2024年）

## プロジェクト概要

本プロジェクトは、2015年から2024年までの日本における物価水準の変化を詳細に分析し、その内訳（食品、穀物、電気、ガス、サービスなど）を体系的にまとめたものです。

## 主要な発見事項（サマリー）

### 総合物価の変化
- **2015年→2024年**: 約11.5%上昇（2020年基準）
- **2024年の前年比**: 2.7%上昇

### 品目別の特徴

| 品目 | 主要な変化 |
|------|-----------|
| 食品 | 4.3%上昇（2024年）、米は64.5%急騰 |
| 電気 | 約89%上昇（2021年2月比） |
| ガス | 45-55%上昇 |
| サービス | 1.3%上昇（緩やか） |

### 3つの時期区分

1. **2015-2019年**: 安定期（年0.3-1.0%の緩やかな上昇）
2. **2020-2021年**: パンデミック期（需要減、一時的なデフレ圧力）
3. **2022-2024年**: 急騰期（エネルギー危機、円安、2-4%台の上昇）

### 累積寄与度（2015-2024年、+11.5ポイント）

上昇の主要因：

| 費目 | 累積寄与度 | 寄与率 |
|------|-----------|--------|
| 食料 | +3.8〜4.2ポイント | 33〜37% |
| 光熱・水道 | +2.5〜3.0ポイント | 22〜26% |
| 教養娯楽 | +1.2〜1.5ポイント | 10〜13% |

**トップ3で約70%を説明**: 食料とエネルギーが物価上昇の主要因

## データの信頼性

### 主要データソース
1. **総務省統計局**: 消費者物価指数（CPI）
2. **日本銀行**: 企業物価指数、サービス価格指数
3. **農林水産省**: 食品価格動向
4. **資源エネルギー庁**: エネルギー価格統計

### データの特性
- **公的統計**: 信頼性が高い
- **2024年データ**: 一部速報値を含む
- **推定値**: 家計影響試算など一部含む
- **更新**: 定期的な統計更新に対応が必要

---

## 分析の限界と注意事項

### 1. データの制約
- 一部詳細データは推定値を使用
- 地域別データは限定的
- 品質変化は完全には反映されない

### 2. 時点の問題
- 2024年データは2024年12月時点までの情報
- 一部速報値が含まれる可能性
- 今後の確報値で修正される場合あり

### 3. 分析の範囲
- 消費者物価を中心とした分析
- 企業物価や資産価格は限定的
- 地域差の詳細は今後の課題
