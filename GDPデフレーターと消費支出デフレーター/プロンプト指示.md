# プロンプト指示

## タスク概要

GDPデフレーターと消費支出デフレーターの長期推移を比較分析し、視覚化とプレゼン資料を作成する。

---

## 指示1: グラフ作成

### 元のプロンプト
```
サブフォルダ「GDPデフレーターと消費支出デフレーター」を作成し、
２つのデフレーターの長期推移をグラフにしてください。1994年か、その前から
```

### 実装内容

1. **サブフォルダ作成**
   - フォルダ名: `GDPデフレーターと消費支出デフレーター`

2. **データ範囲**
   - 期間: 1980年〜2024年（45年間）
   - 1994年よりも前の1980年から開始
   - 基準年: 2015年=100

3. **グラフの種類**（4つのサブグラフ）
   - **グラフ1**: 長期推移（1980-2024）
     - GDPデフレーターと消費支出デフレーターの推移
     - 時代区分の背景色分け（バブル経済、失われた20年、アベノミクス、コロナ後）

   - **グラフ2**: 前年比変化率
     - 年間成長率の推移
     - 2%目標ラインの表示

   - **グラフ3**: 1980年からの累積変化
     - 累積的な物価変動の可視化
     - 最終値の注釈付き

   - **グラフ4**: デフレーター格差
     - 消費支出デフレーター - GDPデフレーターの差
     - プラス/マイナスで色分け

4. **出力ファイル**
   - `deflator_comparison.png` - 英語版
   - `deflator_comparison_jp.png` - 日本語版
   - `create_deflator_comparison.py` - 生成スクリプト

5. **データソース**
   - 内閣府「国民経済計算年次推計」
   - 実際の歴史データに基づく

## 指示2: プレゼン資料

### 元のプロンプト
```
プレゼン資料は、先生と学生の会話形式にする
```

### 実装内容

1. **形式**
   - 先生と学生の対話形式
   - Markdown形式で作成

2. **構成**（全9部構成）
   - 第1部: デフレーターとは何か（基本概念の説明）
   - 第2部: 長期推移から見る日本経済（1980-2024年）
     - バブル経済期（1980年代後半〜1991年）
     - 失われた20年（1991-2013年）
     - アベノミクス期（2013-2020年）
     - コロナ後のインフレ期（2020-2024年）
   - 第3部: 前年比変化率で見る物価変動
   - 第4部: 1980年からの累積変化
   - 第5部: 2つのデフレーターの「格差」
   - 第6部: なぜ2つのデフレーターに差が生まれるのか
   - 第7部: データから読み取れる日本経済の課題
   - 第8部: まとめと今後の展望
   - 第9部: 学生からの質問コーナー

3. **教育的要素**
   - 学生の質問を通じて自然に疑問を提示
   - 先生が段階的に解説
   - グラフを参照しながら説明
   - 実生活との関連づけ

4. **補足情報**
   - 用語集
   - データ出典
   - さらに学びたい人へのリソース

5. **出力ファイル**
   - `プレゼン資料.md` - 完全版プレゼンテーション

## 技術仕様

### Pythonスクリプト
- matplotlib を使用した可視化
- 複数のサブプロットで構成
- 英語版と日本語版の2バージョン生成
- 高解像度出力（dpi=300）

### データ構造
- 年次データ（1980-2024年）
- 2つの時系列データ:
  - GDPデフレーター（2015=100）
  - 消費支出デフレーター（2015=100）

### グラフ設定
- フィギュアサイズ: 16 x 12インチ
- 2行2列のサブプロット配置
- カラースキーム: #2E86AB（GDP）、#A23B72（消費）
- グリッド、凡例、注釈付き
